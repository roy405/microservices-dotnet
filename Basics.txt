Docker:

build an image -> docker build -t <dockerusername>/<servicename> . 
		ex: docker build -t raddish405/platformservice

run image -> docker run -p <port> -d <dockerusername>/<servicename>
		ex: docker run -p 8080:80 -d raddish405/platformservice

check running containers -> docker ps

stop running container -> docker step <container-id>

restart docker id -> docker start <container-id>
